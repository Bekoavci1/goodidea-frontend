# GOODIDEA-FRONTEND

# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version 14
- Install [Yarn](https://yarnpkg.com/)


# Getting started
- Clone the repository
```
git clone https://github.com/Bekoavci1/goodidea-frontend.git
```
- Install dependencies
```
cd good-idea
yarn install
```
- Build and run the project
```
npx expo start
```
